
# Usage

`NOTE: First run server i.e, main.py`

## Backend
- Create virtual environment for python
```bash
cd backend-flask
```
```bash
pip install -r requirements.txt
```

- Start server
```bash
python main.py
```

## Frontend
- cd to `root directory` then execute the below commands
```bash 
npm install
```
```bash
npm run start 
```
